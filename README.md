## Simple Interest Calculator

A clean and user-friendly **Simple Interest Calculator** built using **HTML, CSS, and JavaScript**. This project allows users to calculate the interest earned and total amount based on principal, rate of interest, and time.

## Features

* Easy-to-use interface
* Interactive rate selector (slider)
* Year selection with datalist
* Instant calculation using JavaScript
* Displays:

  * Interest earned
  * Final amount
  * Future year

## Technologies Used

* HTML – Structure of the web page
* CSS – Styling and layout
* JavaScript – Logic and calculations

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100
